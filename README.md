## Run Locally

Open `index.html` directly in a browser.

Or run a local static server:

```bash
python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

No package installation is required.

## Deployment

This portfolio is deployed on Vercel.

To deploy again with Vercel CLI:

```bash
npx vercel --prod
```

You can also deploy from the Vercel dashboard by importing the GitHub repository and selecting the project root. Since this is a static site, no build command is required.
